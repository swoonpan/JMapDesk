Über JMapDesk
*************
JMapDesk ist ein GUI-Frontend zum UMN MapServer auf Basis shp2img (OGR) zur lokalen Anwendung auf Desktops unter Windows (win32) und Linux (x86 32Bit).


Features
********
* Konfigurieren: Pfad zu shp2img.exe wählen
* Datei (.map) wählen (.map ist das Steuerfile von mapserver)
* Zoom in/out, Zoom to full extent, Pan
* Refresh/Reload (wenn .map geändert wurde)
* Legende/Layer darstellen sowie Layer on/off schalten
* Koordinaten darstellen
* Weitere, z.B.
      o Screenshot des Kartenausschnitts.
      o Kartenausschnitt als JPG/GIF/PNG speichern.
      o Map Generator (Beta).
